# ZenGarden

A web-based project designed to provide a calm and interactive user experience.

## Overview
ZenGarden is a front-end web application built with HTML, CSS, and JavaScript. It focuses on clean design and smooth user interactions.

## Features
- Responsive layout for different screen sizes
- Interactive elements powered by JavaScript
- Modular CSS for easy styling updates

## File Structure
- `index.html` – Main structure of the site
- `style.css` – Visual styling and layout rules
- `script.js` – Handles interactive logic
- `assets/` – Directory for images and icons

## How to Run
1. Clone this repository to your local machine
2. Open `index.html` in any modern web browser (Chrome, Firefox, Edge)
